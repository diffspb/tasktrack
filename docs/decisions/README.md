## Шаблон

```markdown
# ADR-XXX: Название

**Статус:** Принято / На рассмотрении / Отклонено  
**Дата:** YYYY-MM-DD

## Контекст
Почему вопрос вообще встал. Какие варианты рассматривались.

## Решение
Что именно решили.

## Последствия
Что это означает для разработки, какие ограничения, что потребует пересмотра в v2.
```
